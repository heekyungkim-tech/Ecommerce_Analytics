# Ecommerce_Analytics_Dashboard
## 기능
- 사용자 로그인
- RFM 활용 고객 세분화
- 고객 데이터 분석 / 판매자 매출 데이터 분석
- 지도 시각화
## 사용
웹 애플리케이션을 실행한 후, 브라우저에서 `http://localhost:5000`을 열어 대시보드를 확인
```bash
    python app.py
```
